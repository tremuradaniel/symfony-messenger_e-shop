# e-shop|PRODUCER (Symfony Messenger)

## Description
- RabbitMQ & CQRS (Command Query Responsibility Segregation) with Symfony Messenger

## Commands

= ```php bin/console messenger:consume``` (consume messages -- to be run from consumer, not producer)
